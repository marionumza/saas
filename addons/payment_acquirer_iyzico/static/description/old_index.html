Iyzico Payment Acquirer

Iyzico payment acquirer is a bridge between Harpiya and Iyzico payment gateway.

Configuration

1- From Accounting: -> Configuraiton -> Payment Acquires


2- From payment acquirers list, choose “Iyzico payment”


3- Enter Iyzico Credentials and other configuration (journal, Messages that wants the customer recieved after finished the payment operation)
