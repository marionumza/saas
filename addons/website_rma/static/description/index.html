Website RMA - Return Orders Management

Website RMA - Return Orders module is used to allow your customer to manage return of the products & create return RMA order from the website.

Features

Manage RMA Request from Website.

Now user can easily manage RMA order from website.

Approve/Reject RMA Request.

User can approve or reject RMA request.

Auto Generate Delivery Order.

Delivery order will generated automatically for RMA order.

Quick Access RMA.

User can quickly access RMA order form sales order and customer form view.
Demo Request Video Tutorial

Return Order

You can return order from Portal/ My Account page. In sale order portal page, Click on Return button to return order.

Return Order Wizard

After click on return button one wizard will open, In that wizard you can pass Return Product,Return Quantity and Return Reason for that order.

Thank You Message

After submitting the return order you will land up to Thank you page and RMA order created at backend.

Return Order Portal View

You can see your return order from my account.

Return Order View

Here you can see all created return order.

Backend - Return Order Tree View
